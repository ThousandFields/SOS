# S.O.S. - Standard Operations Schematics 🛠️⚓

**S.O.S.** is a professional-grade recipe browser and material tracking utility for **Barotrauma**. It provides a seamless, vanilla-like interface to explore crafting, deconstruction, and item dependencies.

## ✨ Key Features
- **Comprehensive Browser:** View Fabrication, Deconstruction, and "Used In" recipes for any item.
- **HUD Tracker:** Track ingredients in real-time with an on-screen checklist.
- **Favorites System:** Keep your most-used items at the top of the list.
- **Smart Navigation:** Browser-like history (Back/Forward) with keyboard shortcuts.
- **Optimized Performance:** Infinite scroll support for huge modpacks (Neurotrauma, etc.).
- **Multi-language:** Fully localized in English and Spanish.

## ⌨️ Controls
- **[J]**: Open / Close the SOS Menu.
- **[Backspace]**: Navigate to previous item.
- **[Ctrl + Backspace]**: Navigate to next item.
- **[Right Click]**: Open context menu (Track, Favorite, etc.).
- **[Escape]**: Close window.
